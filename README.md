## 📝 Descripción del proyecto

**Comment Wall App** es una aplicación web interactiva diseñada para el consumo de una API REST. Permite a los usuarios publicar mensajes, visualizar hilos de conversación en orden cronológico y gestionar el contenido mediante la eliminación de entradas, todo bajo una experiencia fluida sin recarga de página.

## 🚀 Instalación y ejecución

### 1. Clona el repositorio
```bash
git clone [https://github.com/tu-usuario/comment-wall-app](https://github.com/tu-usuario/comment-wall-app)
```

### 2. Descargar la extensión de Live Server - Ritwick Dey

### 3. Abrir el index.html con esta extensión > "Open with Live Server"

NOTA:
El sistema ya apunta al backend hosteado en vercel automáticamente, por lo tanto funciona sin problema alguno.
De ser necesario y si quiere probar el backend, deberá descargarlo localmente y modificar la variable 
const API_URL < se encuentra en app.js al backend corriendo en su computadora

BACKEND:
